## Formatierungs-Tipps

### Markdown in "Weitere Erklärung"

- `**bold**` → **bold**
- `*italic*` → *italic*
- `\n\n` → Neue Absätze
- Listen mit `-` oder `*`
- Überschriften mit `#`, `##`, etc.

## Import in Google Sheets

1. Markiere die Daten (mit Tab getrennt)
2. Kopiere sie
3. Füge sie in dein Google Spreadsheet ein (Strg/Cmd + V)
4. Google Sheets erkennt automatisch die Tab-Trennung

Alternative:
1. Speichere als CSV-Datei
2. In Google Sheets: Datei → Importieren

## Hinweis zu Datumsformaten

Die Datums-Spalten bleiben beim ersten Import leer. Sie werden automatisch gefüllt, wenn du die Karten lernst.

Format: ISO 8601 (z.B., `2025-11-07T10:30:00.000Z`)
