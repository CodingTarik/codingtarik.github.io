# Tools Directory

This directory is reserved for future utility tools.

## Planned Tools

- [ ] Image Converter
- [ ] Color Palette Generator
- [ ] Code Formatter
- [ ] Markdown Preview
- [ ] Text Diff Checker
- [ ] And more...

## Structure

Each tool should have its own subdirectory with:
- Component file (e.g., `ImageConverter.jsx`)
- Supporting files (utils, styles, etc.)
- README with usage instructions

## Example

```
tools/
  image-converter/
    ImageConverter.jsx
    utils.js
    README.md
```
